Модуль Nodemailer

Модуль Nodemailer упрощает отправку писем с вашего компьютера.

Модуль Nodemailer можно загрузить и установить с помощью npm:

	C:\Users\Your Name>npm install nodemailer

После того как вы загрузили модуль Nodemailer, вы можете включить модуль в любое приложение:

	var nodemailer = require('nodemailer');

Отправить письмо

Теперь вы готовы отправлять электронные письма с вашего сервера.

Используйте имя пользователя и пароль от выбранного поставщика электронной почты для отправки электронной почты. В этом уроке вы узнаете, как использовать свою учетную запись Gmail для отправки электронной почты:

	var nodemailer = require('nodemailer');

	var transporter = nodemailer.createTransport({
	  service: 'gmail',
	  auth: {
		user: 'youremail@gmail.com',
		pass: 'yourpassword'
	  }
	});

	var mailOptions = {
	  from: 'youremail@gmail.com',
	  to: 'myfriend@gmail.com',
	  subject: 'Sending Email using Node.js',
	  text: 'That was easy!'
	};

	transporter.sendMail(mailOptions, function(error, info){
	  if (error) {
		console.log(error);
	  } else {
		console.log('Email sent: ' + info.response);
	  }
	});

Вот и все! Теперь ваш сервер может отправлять электронные письма.

Несколько отправок=>получателей

Чтобы отправить электронное письмо нескольким получателям, добавьте их в свойство «to» объекта mailOptions, разделенное запятой:

Отправить письмо по нескольким адресам:

 
		var mailOptions = {
		  from: 'youremail@gmail.com',
		  to: 'myfriend@mail.ru, myotherfriend@yahoo.com',
		  subject: 'Sending Email using Node.js',
		  text: 'That was easy!'
		}

Отправить HTML

Чтобы отправить текст в формате HTML в свой адрес электронной почты, используйте свойство «html» вместо свойства «text»:

 
			var mailOptions = {
		  from: 'youremail@gmail.com',
		  to: 'myfriend@yahoo.com',
		  subject: 'Sending Email using Node.js',
		  html: '<h1>Welcome</h1><p>That was easy!</p>'
		}
